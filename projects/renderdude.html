This is an implementation of the Ray Tracing Series. The current plans for this project are:

1. Live Rendering using an implementation of ImGui or similar.
2. Noise Reduction using Intel Open Image Denoise.
3. Swap to GPU Rendering with Vulcan.
4. Render imported meshes.

This will take a while, however after that is done I am particularly interested in hair. Blocking and attaching hair to the head properly takes an incredible amount of patience and time. Perfect placement is simultaneously not always guaranteed. One of my long term goals is to find a solution that supports artist create believable and diverse hair styles in easier ways.
